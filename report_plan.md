## Exit Criteria

- Wrkr can run deterministic public sample scans at scale (>500 repos) with pagination-safe org acquisition.
- Repo/org scans can be executed in a way that materializes analyzable content for detectors (or clearly fail-closed with explicit reason).
- Wrkr can generate a single deterministic campaign summary artifact with headline metrics and methodology metadata.
- PR distribution path is complete: relevant PRs receive deterministic comment payloads in `mode=pr`.
- UAT install-path checks (including Homebrew path) are wired into release gating policy.
- Any published claim of "unreviewed production access" is blocked or downgraded when production targets are not configured.
